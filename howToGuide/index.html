Android Testing
How-To Guide

This guide will show you how to test an Android application using JUnit and Espresso



Features
This is a how-to guide on testing Android applications. It includes steps on the installation and execution of tests using:
  • JUnit for unit testing
  • Espresso API for integration testing of the UI
Documentation
Where appropriate, this guide will link to official documentation to provide background information.

Requirements
This guide assumes basic knowledge of native Android development in Android Studio, but assumes no experience in testing an Android application.
